- `one_comp_sublingual.toml` — 舌下模型（ka_buccal=2.0, ka_oral=0.5, F_buccal=0.7, F_oral=0.3）

- `validation/observed_iv.csv` — 验证数据

### demo.html + GitHub Pages

### 重要教训
- 时间单位必须统一！OralModule 参数单位是小时，调用方必须用小时
- 一房室模型对快速消除药物在长时间点有局限（需要二房室）
- Shadow Pipeline 验证：CL=20, Vd=50 时平均偏差 15.64%
- **口服 F=1.0 是错误假设**：真实口服药物有首过效应，F 通常 0.3~0.7
- 舌下 vs 口服对比：F_oral=1.0 时舌下"更快但不更强"（反经验），改为 F_oral=0.5 后符合药代预期
- 戊酸雌二醇口服 F≈0.04~0.05（95%首过清除），透皮 F≈0.10~0.15
- **人咪偏好**：不把思考过程发出来

### 给药协议 phases 格式
```toml
[[dosing_protocol.phases]]
n_doses = 2
dose_amount = 4.0
interval = 24.0
```
dosing-parser.ts 支持累积时间偏移，自动处理负荷→维持过渡。
