# 自动对话记忆系统

## 功能

自动将每轮对话缩减为结构化记录,存入每日记忆文件。

## 记录格式

```markdown
## [HH:MM] 对话摘要

- **时间**: YYYY-MM-DD HH:MM
- **人物**: 人咪 / Mon3
- **地点**: QQ私聊 / Telegram / 其他
- **事情**: [一句话概括对话内容]

**详细**:
- [关键信息1]
- [关键信息2]
- ...
```

## 自动提取规则

### 时间
- 自动从消息元数据提取
- 格式: YYYY-MM-DD HH:MM

### 人物
- 人咪 (用户)
- Mon3 (猫/AI)
- 其他人 (如果有)

### 地点
- QQ私聊: `qqbot:c2c:{id}`
- Telegram: `telegram:{chat_id}`
- 其他渠道自动识别

### 事情
- 提取对话主题/意图
- 一句话概括(20字以内)
- 标记重要决策/信息

## 实现方式

在每次对话后,自动调用此模块:
1. 解析对话内容
2. 提取关键信息
3. 格式化写入memory文件
4. 避免重复记录

## 示例

```markdown
## [13:40] 配置心跳频率

- **时间**: 2026-03-24 13:40
- **人物**: 人咪 / Mon3
- **地点**: QQ私聊
- **事情**: 将心跳频率从30分钟改为45分钟

**详细**:
- 修改 openclaw.json 配置
- 添加 agents.defaults.heartbeat.every = "45m"
- 需要重启Gateway生效
```
